# Casino Royale
Casino Online, con una ruleta individual y un BlackJack multi-jugador en tiempo real con sockets
- Web FrameWork: Express.JS
- Base de Datos: MySQL
- Motor Generador de Vistas: Handlebars
- Manejo de sockets: Socket.io
